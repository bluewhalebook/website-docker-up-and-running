The 3rd edition is here!

The Early Release of the 3rd edition of Terraform: Up & Running is here! The 3rd edition adds about 100 pages of new content, including two totally new chapters, plus major updates to all the existing chapters:

  • New chapter: how to manage secrets with Terraform (KMS, Vault, OIDC, etc)
  • New chapter: working with multiple regions, accounts, and clouds (including Kubernetes!).
  • All code examples have been updated from Terraform 0.12 to 1.1.
  • Provider management with required_providers blocks and the provider lock file.
  • Variable validation and sensitive variables.
  • Refactoring with moved blocks.
  • Using count and for_each with module blocks.
  • Policy enforcement using OPA.
  • And much more! Check out the announcement blog post for the full details.
